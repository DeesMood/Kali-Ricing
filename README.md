## KALI RICING


**1. Wallpaper**
* Move "kali-wallpaper.png" to "/usr/share/backgrounds/kali/"
* Desktop (Settings) > "Choose the wallpaper"


**2. Restore Panel Backup**
* Download "Panel.tar.bz2"
* Panel Profiles > Import > Apply "Backup\_..."


**3. Use GTK**
* Download and move "Graphite-teal-Dark-nord (GTK)" to "/usr/share/themes/"
* Apperance > Style > Select "Graphite-teal-Dark-nord (GTK)"


**4. Icon is default from kali "Flat-Remix-Blue-Dark"**


**5. Use monospace font**
* Apperance > Fonts > Select "Monospace Regular"
